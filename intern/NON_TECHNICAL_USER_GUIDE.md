## üéØ What is a Mindmap?

A **mindmap** is a visual way to organize ideas. Think of it like a tree:
- The **root** (top) is your main topic
- **Branches** (below) are related ideas
- **Leaves** (further down) are details

**Example**: Planning a party
- Root: "My Birthday Party"
- Branch: "Food" ‚Üí Leaves: "Cake", "Pizza", "Drinks"
- Branch: "Guests" ‚Üí Leaves: "Family", "Friends"
- Branch: "Location" ‚Üí Leaves: "Home", "Park"

## ‚úèÔ∏è CRUD Operations - Simple Guide

**CRUD** stands for:
- **C**reate (Add new nodes)
- **R**ead (View nodes)
- **U**pdate (Edit nodes)
- **D**elete (Remove nodes)

### üìù CREATE - Adding New Nodes

#### Method 1: Using the Green Button (Easiest!)
1. **Click** a node to select it (it gets a blue border)
2. A **green button with a +** appears on the node
3. **Click the green button**
4. A new child node appears below!

#### Method 2: Right-Click Menu
1. **Right-click** any node
2. A menu appears
3. Choose **"Add Child"** or **"Add Sibling"**
4. New node appears!

#### Method 3: Toolbar Button
1. Click the **+ button** in the top toolbar
2. Adds a new node to the root

**What's the difference?**
- **Child**: Goes under the selected node
- **Sibling**: Goes next to the selected node (same level)

---

### üëÅÔ∏è READ - Viewing Nodes

#### View Node Details
1. **Click** any node
2. Details appear in the right panel
3. You'll see:
   - Full title
   - Summary
   - Description
   - Tags (if any)
   - Children count

#### Expand/Collapse Nodes
- **Click the arrow button** on a node
- **Arrow down** = Expanded (children visible)
- **Arrow right** = Collapsed (children hidden)

**Why collapse?**
- Keeps large mindmaps organized
- Focus on one section at a time
- Easier to navigate

#### Search for Nodes
1. **Type in the search box** at the top
2. Matching nodes appear in a list
3. **Click a result** to go to that node
4. The app automatically shows it to you!

---

### ‚úèÔ∏è UPDATE - Editing Nodes

#### Quick Edit (Title Only)
1. **Double-click** the node title
2. Title becomes editable
3. Type your changes
4. Press **Enter** to save
5. Press **Escape** to cancel

#### Full Edit (Everything)
1. **Click the edit button** (pencil icon) on a node
2. Edit dialog opens with two tabs:

   **Basic Tab:**
   - Title
   - Summary (short description)
   - Description (longer details)

   **Metadata Tab:**
   - Status (draft, completed, important, archived)
   - Tags (add multiple tags)
   - Custom Fields (add your own information)

3. Make your changes
4. Click **"Save Changes"**

**Tips:**
- Summary shows in tooltips (hover over node)
- Description shows in the side panel
- Tags help organize and find nodes later

---

### üóëÔ∏è DELETE - Removing Nodes

#### Method 1: Using the Red Button
1. **Click** a node to select it
2. A **red button with trash icon** appears
3. **Click the red button**
4. Confirm deletion in the popup
5. Node is removed (including all its children!)

#### Method 2: Right-Click Menu
1. **Right-click** a node
2. Choose **"Delete"**
3. Confirm deletion

**‚ö†Ô∏è Important Warning:**
- Deleting a node **also deletes all its children**
- This cannot be undone (unless you use Undo)
- The root node cannot be deleted

## üéì Step-by-Step Examples

### Example 1: Creating a Simple Mindmap

**Goal**: Plan a vacation

1. **Start**: You see the root node "Software Development"
2. **Double-click** the title, type "My Vacation Plan"
3. **Click** the root node to select it
4. **Click the green + button** ‚Üí "New Child Node" appears
5. **Double-click** "New Child Node", type "Destination"
6. **Click** "Destination" node
7. **Click green + button** ‚Üí Add "Beach" as child
8. **Click green + button** again ‚Üí Add "Mountains" as child
9. **Repeat** for "Budget", "Dates", etc.

**Result**: You have a complete vacation planning mindmap!

---

### Example 2: Organizing with Tags

**Goal**: Organize work tasks

1. **Create nodes** for your tasks
2. **Click edit button** on a task node
3. **Go to Metadata tab**
4. **Add tags**: Type "urgent" and press Enter
5. **Add more tags**: "work", "important"
6. **Set status**: Choose "Important" from dropdown
7. **Save**

**Result**: Tasks are now organized and easy to find!

---

### Example 3: Using Search

**Goal**: Find a specific node in a large mindmap

1. **Type** in the search box (e.g., "budget")
2. **See results** appear in dropdown
3. **Click** the result you want
4. **App automatically**:
   - Expands parent nodes
   - Zooms to the node
   - Selects it for you

**Result**: Found instantly, no scrolling needed!

## üí° Tips for Non-Technical Users

### ‚úÖ Do's

1. **Start Simple**: Begin with 3-5 nodes, add more later
2. **Use Tags**: They help organize and find things
3. **Collapse Branches**: Keep things tidy
4. **Save Often**: It saves automatically, but you can click Save button
5. **Use Search**: Faster than scrolling
6. **Double-click Titles**: Quick way to rename
7. **Right-click**: Always shows more options

### ‚ùå Don'ts

1. **Don't Delete Carelessly**: Remember, children are deleted too!
2. **Don't Make Too Deep**: 4-5 levels is usually enough
3. **Don't Skip the Welcome Guide**: It's helpful!
4. **Don't Worry About Mistakes**: Use Undo (Ctrl+Z)

## üÜò Common Questions

### Q: How do I add a node?
**A**: Click a node, then click the green + button that appears.

### Q: How do I change a node's name?
**A**: Double-click the title, type new name, press Enter.

### Q: How do I delete a node?
**A**: Click it, then click the red trash button.

### Q: I made a mistake! How do I undo?
**A**: Click the undo button (‚Ü∂) in the toolbar, or press Ctrl+Z.

### Q: Where did my node go?
**A**: 
- Check if parent is collapsed (click arrow to expand)
- Use search to find it
- Check if it was deleted (use Undo)

### Q: How do I see more details about a node?
**A**: Click the node, details appear in the right panel.

### Q: Can I move nodes around?
**A**: Yes! Click and drag nodes to reposition them.

### Q: How do I save my work?
**A**: It saves automatically! But you can click the Save button too.

### Q: Can I change colors?
**A**: Colors are automatic based on level, but you can customize in advanced settings.

### Q: What if I get lost?
**A**: 
- Click "Fit to View" button (bottom right)
- Use search
- Check the mini-map (bottom left)
